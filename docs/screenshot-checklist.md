# Screenshot Checklist

Use this checklist to confirm that the Microsoft 365 User Lifecycle Management project shows both manual admin center workflows and automation results.

## Admin Center Workflows

- User profile before password reset
- Password reset window
- Password reset confirmation
- Block sign-in action
- Blocked sign-in status
- Unblocked account
- User license tab
- Users and assigned licenses view

## Automation Evidence

- Bulk onboarding HTML summary
- Lifecycle audit summary before fixes
- Lifecycle remediation preview mode
- Remediation PowerShell output after fixes
- Lifecycle remediation summary after fixes
- Lifecycle audit summary after fixes
- Master dashboard

## Review Notes

Before publishing screenshots, confirm that:
- No production tenant is shown
- No real employee data is shown
- No production passwords are shown
- Demo tenant information is clearly lab-only
- Screenshots match the README descriptions
